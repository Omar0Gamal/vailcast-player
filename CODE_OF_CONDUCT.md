# Code of Conduct

## Our Pledge

We are committed to making participation in this project a harassment-free experience for everyone.

## Expected behavior

- Be respectful and constructive.
- Assume positive intent.
- Focus on technical merit and shared goals.

## Unacceptable behavior

- Harassment, discrimination, or hateful conduct
- Personal attacks or insults
- Publishing private information without permission

## Reporting

Report unacceptable behavior by opening a private maintainer contact through GitHub.

Project maintainers may remove, edit, or reject comments, commits, code, or issues that violate this Code of Conduct.
