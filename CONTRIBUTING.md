# Contributing

Thanks for your interest in contributing to vailcast-player.

## Prerequisites

- Node.js 18+
- npm 9+

## Setup

1. Fork and clone the repository.
2. Install dependencies:
   - `npm install`
3. Run checks:
   - `npm run validate`

## Development workflow

1. Create a branch from `main`.
2. Keep changes focused and small.
3. Add tests for behavior changes.
4. Run `npm run validate` before opening a PR.
5. Open a pull request using the PR template.

## Commit conventions

Use Conventional Commits where possible.

Examples:

- `feat(core): add playback event callbacks`
- `fix(react): prevent duplicate player initialization`
- `docs: clarify Angular setup`

## Reporting issues

Use the issue templates for bug reports and feature requests.
